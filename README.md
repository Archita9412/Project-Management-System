# Project Management System

A scalable full-stack Project Management System built to streamline collaboration, task tracking, and project organization. The application provides a structured environment where users can manage projects, assign tasks, track progress, and maintain project-related data efficiently.

The system is designed using a modular backend architecture, ensuring maintainability, scalability, and clean separation of concerns.

---

## Overview

This project focuses on solving common challenges in team collaboration by providing:

- Centralized project management
- Structured task hierarchy (tasks and subtasks)
- Secure authentication and authorization
- Efficient data handling with a NoSQL database
- Extensible backend design for future integrations

---

## Features

- User authentication and authorization using JWT
- Project creation, update, and deletion
- Task and subtask management
- File upload and storage handling
- Project notes management
- RESTful API architecture
- Error handling and middleware-based request processing

---

## Tech Stack

### Backend
- Node.js (runtime environment)
- Express.js (web framework)

### Database
- MongoDB (NoSQL database)
- Mongoose (ODM for schema modeling)

### Authentication & Security
- JSON Web Tokens (JWT)
- bcrypt (password hashing)
- dotenv (environment variable management)

### File Handling
- Multer (multipart/form-data handling)

### Development Tools
- Nodemon (development server auto-restart)
- Git (version control)
- GitHub (code hosting)

### API & Architecture
- RESTful API design
- MVC (Model-View-Controller) architecture
- Middleware-based request lifecycle
